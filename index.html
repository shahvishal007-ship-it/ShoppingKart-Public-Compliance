Current Alpha boundaries

The submitted closed-test build is deliberately limited while the production backend and live integrations are completed.

  • No live retailer APIs or scraping
  • No checkout, payment or affiliate transaction
  • No advertising or Advertising ID use
  • No off-device collection of required user data in this build

What comes later

Production account services, pincode serviceability, optional location assistance, operational admin controls and legitimate provider integrations remain future launch work.
