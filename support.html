Support

Help for approved closed-test users, reviewers and people with privacy or deletion questions.

Email support

Use a clear subject and include only information needed to diagnose the issue.

shahvishal007@gmail.com

Before contacting support

  • Confirm that you are using the Google account approved for the closed test.
  • Note the screen where the issue occurred and the action immediately before it.
  • Include the Android version and device model when reporting a technical issue.
  • Do not send passwords, OTPs, payment details or unnecessary personal information.

Recommended email format

Subject: TOLZA closed-test issue — [short description]
Include: device, Android version, app screen, expected result, actual result and a screenshot without private information.

Common support topics

I cannot install the closed test

Make sure the Google account used on the Play Store is included in the approved tester list and that you joined through the tester opt-in link.

The synthetic login does not work

Use only the test instructions provided for the active closed-test release. Do not use personal credentials unless the release instructions explicitly require them.

I found an incorrect comparison

The current build uses synthetic data. Please report the exact basket and screen so the deterministic comparison behaviour can be checked.

I have a privacy or deletion request

Use the Privacy Policy and Account Deletion pages, or email support with the appropriate subject.

Availability

Support is provided on a reasonable-effort basis during closed testing. TOLZA is not yet a live commerce or emergency service.
